# ✈️ Travel Agent Chatbot (LangChain + Groq)

## 📌 Features
- Travel-only chatbot
- Rejects non-travel queries
- Streaming responses (token-by-token)
- Manual memory using Python list
- Auto summarization every 5 turns

## 🛠 Tech Stack
- LangChain
- Groq (LLM)
- Python
